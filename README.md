# CLI Calculator (TS, Node.js)

### How to run

- Use Node v18

```cmd
nvm use
```

- Install dependencies

```cmd
npm i
```

- Run the node app

```cmd
npm start
```

- You can run as a CLI using

```cmd
npm link
```

- And then

```cmd
calc-assignment
```
